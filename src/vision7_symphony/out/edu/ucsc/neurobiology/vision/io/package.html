This package implements the Vision IO Pipeline (VIOP).

TheVIOP is based on the following classes:
AsynchronousInputStream
AsynchronousOutputStream
SampleInputStream
CompressedSampleInputStream
MultipleCompressedSampleInputStream
SpikeSaver

The package also contains a series of independent classes used as intefaces to high level data files:
ClusteringModelFile,
CovarianceFile,
NeuronFile,
ProjectionsFile,
ParametersFile,
SpikeFile,
STAFile,
PhysiologicalImagingFile,
RawMovieFile,
HistogramFile
